info About

Red CSS is a simple CSS library inspired by Bootstrap

Red CSS is Red Script's younger brother, its also the more stylish brother

Red CSS is the second oldest library in the Red Code Family (it goes Red Script, Red CSS, Red HTML, Red logos NOTE: Red Logos is Discontinued and is used by Red Icons info-circle , Red Icons, Red X, Red Python, Red Java, Red C,Red Plus, and Red Sharp)

Features

Red CSS has many helpful CSS presets



Bold,Italic,Underline

Red CSS has the Bold,Italic, and Underline, Tags,Classes, and Attributes
Tags: <bold></bold>,<italic></italic>,<underline></underline>
Classes:<div class="bold italic underline"></div>
Attributes:<div bold italic underline></div>

PJO,pjo,CHB,chb

Tags:<pjo></pjo> or <PJO></PJO>,<CHB></CHB> or <chb></chb>
Classes: <div class="PJO(sam as pjo) CHB(same as chb)"</div>
Attributes:<div PJO (or pjo) CHB (or chb)></div>
